# 🌐 Plantilla Web de Fundamentos (HTML5 & CSS3)

Este proyecto es una exhibición de mis bases sólidas en el desarrollo frontend. Se trata de una estructura de página completa donde aplico conceptos avanzados de diseño web sin depender de frameworks externos.

---

## 🎨 Características Técnicas
* **HTML5 Semántico:** Uso correcto de etiquetas `<header>`, `<section>`, `<article>` y `<footer>` para mejor estructura y SEO.
* **Layout con Flexbox:** Implementación de un sistema de rejilla flexible para la organización de posts y secciones dinámicas.
* **Estilos Avanzados:** - Gestión de box-model (márgenes, rellenos y bordes).
  - Efectos de transición y pseudo-clases en enlaces y botones.
  - Tipografías personalizadas y jerarquía visual.
* **Código Limpio:** Separación total de estructura (HTML) y presentación (CSS).

---

## 🛠️ Tecnologías
* **HTML5:** Estructura robusta.
* **CSS3:** Diseño, colores y animaciones.
